# Slack 通知設定指南

## 1. Slack 應用程式設定
1. 前往 https://api.slack.com/apps
2. 建立新應用程式 (Create New App)
3. 選擇 'From scratch' 並命名您的應用程式
4. 選擇您的 Slack 工作區

## 2. Bot Token 取得
1. 在應用程式設定頁面，點擊 'OAuth & Permissions'
2. 在 'Scopes' 區塊新增以下權限：
   - channels:read (讀取頻道資訊)
   - chat:write (發送訊息)
   - chat:write.public (在公開頻道發送訊息)
3. 點擊 'Install to Workspace'
4. 複製 'Bot User OAuth Token' (以 xoxb- 開頭)

## 3. 邀請 Bot 到頻道
1. 在 Slack 中輸入: /invite @您的機器人名稱
2. 或在頻道設定中手動邀請

## 4. 團隊成員通知設定

### 方法一：Slack 手機 App 推播 ⭐ **推薦**
- 確保 Slack App 已開啟推播通知
- 在設定 > 通知 中調整通知偏好
- **優點**：即時性高，團隊成員可立即回應

### 方法二：Email 轉寄
- 在 Slack App 中：設定 > 通知 > 轉寄通知到 email
- 設定特定關鍵字觸發 email 通知
- **適用場景**：離線時仍需接收重要通知

### 方法三：第三方整合工具
- 使用 Zapier 或 IFTTT 連接 Slack 到其他服務
- 設定規則：當 Slack 收到特定訊息時，觸發 email 或 SMS
- **適用場景**：需要複雜自動化流程

## 5. 通知流程說明

### 收到 Slack 通知後的工作流程：

1. **立即確認**：在 Slack 中快速確認通知內容
2. **分配任務**：如果需要多人處理，在 Slack 中 @相關人員
3. **記錄處理**：在同一個對話串中記錄處理進度
4. **關閉通知**：處理完成後標記為已解決

### 通知範例
當系統偵測到重要事件時，會自動發送類似以下訊息：

```
🚨 新租客預約通知
📍 物件：信義區 3 房 2 廳
👤 租客：張小姐
📅 時間：2026-01-15 14:00
📞 聯絡方式：0912345678

@客服團隊 請確認預約並聯繫租客確認細節
```

### 緊急程度分類：
- 🚨 **緊急**：系統錯誤、資料遺失
- ⚠️ **重要**：新預約、逾期提醒
- ℹ️ **一般**：狀態更新、例行報告

## 6. 最佳實務

1. **設定通知關鍵字**：為不同類型事件設定特定 emoji 或關鍵字
2. **建立專用頻道**：不同類型通知使用不同頻道
3. **設定靜音時間**：避免非工作時間被打擾
4. **定期檢討**：每週檢討通知的有效性和必要性

這樣您就能在 Slack 收到通知後，立即知道該採取什麼行動了！
