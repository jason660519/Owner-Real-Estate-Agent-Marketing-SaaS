# 房屋資料準備手冊

本手冊旨在協助房東準備出租房屋所需之各項資料，以利 AI 助理快速為您生成精美的房屋行銷網頁與處理後續租賃事宜。

## 1. 房屋照片 (必要)
為了讓您的物件在市場上更具吸引力，請準備 **4 到 20 張** 高清晰度的照片。

### 拍攝建議：
*   **客廳**：從角落拍攝，展現空間感，盡量納入窗戶採光。
*   **臥室**：主臥與次臥皆需拍攝，展示收納空間（如衣櫃）。
*   **廚房**：拍攝整體流理台、爐具與收納櫃。
*   **衛浴**：保持乾燥整潔，拍攝洗手台、馬桶與淋浴區（乾濕分離）。
*   **陽台/窗外景觀**：如有陽台或良好視野請務必拍攝。
*   **公設/外觀**：大樓外觀、大廳、電梯或休閒設施（健身房、泳池）。

### 注意事項：
*   請橫向拍攝 (Landscape)。
*   拍攝前請整理環境，移除個人雜物。
*   光線要充足，白天拍攝效果最佳。

## 2. 房屋基本資料介紹 (必要)
請準備一份簡單文字描述，包含以下資訊：

*   **物件地址**：完整地址（含樓層）。
*   **格局**：幾房、幾廳、幾衛、幾陽台。
*   **坪數**：室內坪數、權狀坪數。
*   **租金期望**：預計每月租金（是否含管理費/車位費）。
*   **押金要求**：通常為兩個月。
*   **樓層/總樓層**：例如：5F / 12F。
*   **特色描述**：例如：近捷運、全新裝潢、可寵物、附傢俱家電等。

## 3. 權利證明文件 (身分核實用)
為了確保出租安全性與合法性，請準備以下文件影本或翻拍照片：

### A. 建物所有權狀 (權狀影本)
*   用途：證明您是房屋的所有權人。
*   請拍攝權狀正面，確保「所有權人姓名」、「建物門牌號碼」清晰可見。

### B. 建物謄本 (選填，但建議提供)
*   用途：提供更詳細的建物資訊（如正確坪數、完工日期等）。
*   可至地政事務所申請，或使用自然人憑證線上申請電子謄本。

### C. 屋主身分證明文件
*   用途：確認申請人身分與權狀相符。
*   **身分證**：正反面影本/翻拍。
*   **健保卡/駕照**：第二證件（視需求提供）。

## 4. 其他輔助資料 (加分項)
*   **傢俱家電清單**：如出租含傢俱，建議列出清單（品牌/狀況）。
*   **水電費帳單樣本**：協助計算平均費用或確認水電號碼（過戶用）。
*   **房屋平面圖**：如有原始平面圖，可幫助租客了解格局。
