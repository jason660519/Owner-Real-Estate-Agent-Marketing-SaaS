# Frontend - Landlord SaaS Dashboard

This project is built with **Expo** and **React Native**, compatible with iOS, Android, and Web.
The UI is modeled after the TurboTenant dashboard.

## Getting Started

1.  **Install dependencies**:
    ```bash
    npm install
    ```

2.  **Run on Web**:
    ```bash
    npm run web
    ```
    Open [http://localhost:8081](http://localhost:8081) in your browser.

3.  **Run on iOS/Android**:
    ```bash
    npm run ios
    # or
    npm run android
    ```

## Project Structure

- `App.tsx`: Entry point.
- `src/Dashboard.tsx`: Main dashboard layout and widgets.
- `src/components/Sidebar.tsx`: Navigation sidebar (visible on desktop/web).

## Features

- Responsive Dashboard Layout
- Widget Components (Vacancy, Residency, Maintenance)
- Quick Actions Menu
- Sidebar Navigation
