## 主要更新

根據專案架構討論，已進行以下更新：

1. **資料庫架構**：
   - 主資料庫：PostgreSQL（使用 JSONB 儲存謄本資料）
   - 分析型資料庫：ClickHouse（可選，用於財務報表）

2. **後端架構**：
   - Backend API：Node.js + Express/Fastify + LangChain
   - Rasa：Python（AI 對話服務）

3. **流程圖更新**：
   - 所有流程圖中的資料庫已更新為 PostgreSQL
   - 添加了 Backend Process 節點
