# Real Estate SaaS Skills Directory

這個目錄包含了房地產 SaaS 專案的專門技能模組。

## 可用技能

### [Property Management](./property-management/SKILL.md)
物業管理系統開發技能，包含物業 CRUD 操作、狀態管理、搜尋過濾和業務邏輯。

**主要功能**:
- 物業數據模型設計
- 前後端 API 整合
- 搜尋和過濾功能
- 權限控制和驗證
- 圖片管理和優化

**適用場景**:
- 創建物業管理組件
- 實現物業搜尋功能
- 設計物業數據模型
- 優化物業查詢性能

### [User Authentication](./user-authentication/SKILL.md)
多角色用戶認證系統，支援業主、代理商、租客、買家四種角色。

**主要功能**:
- JWT token 認證
- 多角色權限控制
- 用戶註冊和登入
- 密碼安全管理
- 權限保護組件

**適用場景**:
- 實現用戶認證流程
- 設計角色權限系統
- 創建登入註冊表單
- 保護敏感頁面和 API

## 使用方式

當您在開發房地產 SaaS 功能時，Claude 會根據您的描述自動建議相關的技能。您也可以直接引用特定技能：

```
請使用 property-management 技能幫我創建一個物業搜尋組件
```

## 技能擴展

如需添加新的技能模組，請按照以下結構創建：

```
.claude/skills/[skill-name]/
└── SKILL.md
```

每個技能的 `SKILL.md` 檔案應包含：
- 技能描述和核心概念
- 代碼示例和最佳實踐
- 相關的型別定義和 API 設計
- 使用場景和注意事項

## 房地產領域特定考量

這些技能都專門針對房地產業務需求設計，包含：

- **多租戶架構**: 支援業主、代理商、租客、買家
- **地理位置查詢**: PostGIS 和地理索引優化
- **複雜搜尋過濾**: 價格、位置、房型、設施等多維度搜尋
- **業務工作流**: 租賃流程、續約、終止等狀態管理
- **法規合規**: 房地產相關法律要求和數據保護
- **性能優化**: 大量物業數據的查詢和展示優化
