# 租賃資訊 | Leasing Information

## 成為 Australia 108 的尊貴住戶

這是一個難得的機會，入住墨爾本最具代表性的地標建築。我們歡迎優質租客申請，體驗雲端之上的生活。

### 租賃詳情 (Leasing Details)
- **房型**：2 房 2 衛
- **車位**：請洽詢
- **租期**：12 個月起 (可議)
- **入住時間**：即刻入住 (Available Now)

### 申請流程 (How to Apply)
1. **預約看房**：請聯繫下方代理人安排私人導覽或參加公開看房 (Open Inspection)。
2. **提交申請**：滿意後，請填寫租賃申請表並提供身份證明與收入證明文件。
3. **簽署合約**：申請通過後，簽署維多利亞州標準租賃合約 (Residential Tenancy Agreement)。
4. **入住檢查**：入住前將進行詳細的屋況檢查 (Condition Report)，確保雙方權益。
