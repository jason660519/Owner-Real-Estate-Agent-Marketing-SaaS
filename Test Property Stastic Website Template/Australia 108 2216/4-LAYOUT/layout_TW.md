# 室內格局 | Interior & Layout

## 現代奢華與實用美學的完美演繹

這套精心設計的 **2房2衛** 公寓，以開放式格局最大化空間感與採光，每一個細節都流露出現代都市的精緻品味。

### 開放式起居空間 (Living & Dining)
- **全景落地窗**：將壯闊的城市美景引入室內，白天採光充足，夜晚盡享百萬夜景。
- **寬敞客廳**：鋪設高品質木地板，連接用餐區，是家庭聚會或放鬆休憩的理想場所。

### 現代化廚房 (Gourmet Kitchen)
- **頂級配備**：配備高品質 Miele 廚電系列（烤箱、瓦斯爐、洗碗機）。
- **石材中島**：優雅的石材檯面與防濺板，搭配充足的收納櫃體，兼具美觀與機能。

### 舒適臥室 (Bedrooms)
- **主臥室 (Master Suite)**：擁有絕佳景觀，附設專屬衛浴 (Ensuite) 與寬敞的內建衣櫃 (Built-in Robes)。
- **次臥室 (Second Bedroom)**：空間方正，採光良好，適合做為客房或居家辦公室。

### 精緻衛浴 (Bathrooms)
- **時尚設計**：採用現代化衛浴設備，搭配大尺寸鏡面與收納櫃。
- **乾濕分離**：淋浴間採乾濕分離設計，鋪設防滑地磚，既安全又易於清潔。
