# 核心地段 | Southbank

## 置身墨尔本的艺术与娱乐心脏

**Australia 108** 坐落于墨尔本充满活力的 **Southbank** 区，这里是城市文化、艺术与顶级娱乐的交汇点。居住于此，您将享受到无与伦比的便利与丰富的生活方式。

### 步行生活圈 (Walk Score: 99)

- **交通枢纽**：步行即可抵达 Flinders Street Station，轻松连接墨尔本各地。
- **购物天堂**：紧邻墨尔本 CBD 购物区、Collins Street 精品店。
- **美食飨宴**：Southbank Promenade 拥有众多世界级餐厅、河畔咖啡馆与酒吧。

### 文化与娱乐

- **Crown Casino**：南半球最大的娱乐中心，仅几步之遥。
- **艺术特区**：维多利亚国家美术馆 (NGV)、艺术中心 (Arts Centre) 近在咫尺，随时沉浸于艺术熏陶之中。
- **皇家植物园**：城市中的绿洲，是您晨跑或周末野餐的最佳去处。

住在 Australia 108，整个墨尔本最精华的部分，就在您的脚下。
